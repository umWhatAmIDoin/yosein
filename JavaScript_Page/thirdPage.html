Term / Tag Definition Example / Usage
document.getElementsByTagName() Find and target single HTML tag with specific tag name. To target "p", then use "document.getElementsByTagName("p")"
document.getElementById() Find and target single HTML tag with specific id name. Let's say " p id="context1" " exists, then targets it as "document.getElementById("context1")"
document.getElementsByClassName() Find and target single HTML tag with specific class name. Let's say " p class="context1" " exists, then targets it as "document.getElementsByClassName("context1")"
console.log() Display data in console. Console is not normal website display but rather within coding app itself, where console system exists separately like Visual Studio or Codepen. N/A
alert() Display data with pop-up style. Usually works with interacting HTML elements in website. Shows certain message(s) like "Hello World!" as a focused pop up generally in above, when pressed a certain HTML buttons or something similar.
let Creates variable. Can be changed. let something = true;
var Creates variables. Can be changed, but only use it if certain variable must be supported on older system. var something = "something1";
const Creates variables that stays same consistently. const means Constant. things like PI is unchangeable and constant so it works. e.g. "const PI = 3.14159;"
NaN Means Not a Number. Used for Number data types. N/A
== Equal To ~ (4 == "4") returns as true
=== Equal Value and Equal Type (3 === "3") returns as false
!= Not Equal N/A
!== Not Equal Value or Not Equal Type N/A
&& Logical And "If A is true and B is true", (A && B) returns as true. However if either one or both are false, (A && B) returns as false.
|| Logical Or "If both A and B are false", (A || B) returns as false. Other cases return as true.
! Logical Not "If A is true", (!A) returns as false. Basically an opposite meaning of something.
