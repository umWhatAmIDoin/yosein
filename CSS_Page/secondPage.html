Term / Tag Definition Example / Usage
Selector The tag you target to change. h1 / p / div / form / basically anything
# Used for target HTML id. "p id=context1" in HTML goes to CSS as "#context1 {}"
. Used for target HTML class. "p class=context1" in HTML goes to CSS as ".context1 {}"
Selector: Descendant Target specific applicable element(s) that's within another element(s). It will be like: "Selector1 (this space) Selector2 (this space) ~ " e.g. "body div p {}" targets "p", that's inside of "div", that's also inside of "body".
Selector: Child Target a selector that's directly connected to the another. It will be like: "Selector > Selector {}" e.g. Let's say " div id="context1" " exists that include "ol", "#context1 > ol {}" targets "ol" list that's inside of "div" tag with id name "context1".
Selector: Adjacent Target Selector(s) that's next to the another. It will be like: "Selector + Selector {}" e.g. Let's say "p" exists just below "ul" list, and if want to target both ul and p, use "ul + p {}"
Selector: Pseudo-class Used to cover situations that can't be solved with standard HTML. It will be like: "Selector:special-Pseudo-class-name {}" :hover / :active / :link / :nth-child(n) / :not() / :lang() / and so on.
color / background-color changes the background color of entire website page. Color can be described as either Name (e.g. cyan / red / grey / light-yellow), Hexedecimal (e.g. #rrggbb), RGB (Red, Green, Blue (e.g. 255, 0, 0)) / RGBA (Red, Green, Blue, Alpha), HSL (Hue, Saturation, Lightness (e.g. 146, 45%, 27%)), or HSLA (Hue, Saturation, Lightness, Alpha). (the background color on this page is light yellow if you look closely)
px Stands for pixel, a CSS fixed value measurement. Used as often as viewport series(v). N/A
vw / vh Viewport series. vw is Viewport Width and vh is Viewport Height. Used to change length of width and height of certain component(s). 1 vw is 1% of total width of certain component(s) targetting on, same as 1 vh being 1% of total height. N/A
vmin / vmax Used to change length of width and/or height but based on the device user is using. vmin changes based on the smaller width/height, while vmax is based on the larger width/height. For example, if user uses Laptop/Desktop, the screen height is generally shorter than screen width. In that case, vmin will change width or height based on screen height, while vmax will be based on screen width.
* Used for cover EVERYthing within whole page. Very strong for debugging. N/A
